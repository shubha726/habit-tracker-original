# habit-tracker-original
habit beats
Habit Beats is a web application designed to help users track and build positive daily habits through an intuitive interface, streak visualization, and progress analytics.

Features
Habit Creation & Tracking: Add custom habits, mark completions, and view streaks.

Dashboard Overview: Real-time stats including longest streak, completion rate, and calendar view.

Responsive Design: Works seamlessly on desktop, tablet, and mobile devices.

Data Persistence: Local storage ensures your progress is saved across sessions.

Goal Setting: Set targets and receive motivational feedback.
