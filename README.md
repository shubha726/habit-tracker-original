# habit-beatz
Habit Beats is a web application designed to help users track and build positive daily habits through an intuitive interface, that allows the user to watch their favourite videos or listen to music on youtube after the task assigned is completed.

Features
Habit Creation & Tracking: Add custom habits, mark completions, and view streaks.

Responsive Design: Works seamlessly on desktop and mobile devices.

Data Persistence: Local storage ensures your progress is saved across sessions.Embeded youtube playlist links won't work.

Goal Setting: Set targets and receives motivational rewards.
